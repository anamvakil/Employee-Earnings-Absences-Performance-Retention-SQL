## Viewing the SQL Scripts

Each SQL file in this directory can be opened directly on GitHub for review.

To view a script:
1. Click on the file name (e.g., `01_Create_Tables.sql`)
2. GitHub will display the full SQL script in the browser
3. Scripts are ordered numerically to reflect the intended execution flow
